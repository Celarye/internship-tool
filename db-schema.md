# Internschip Tool

Tool voor het beheer van stages:

## Features

- Aanmelden.
  - Door gebruikers.
- Registreren van bedrijven.
  - Door bedrijf medewerkers.
- Indienen stagevoorstellingen.
  - Door bedrijf medewerkers.
- Goedkeuren van stagevoorstelling.
  - Door beheerders.
- Toekenen goedgekeurde stagevoorstellingen aan studenten.
  - Door beheerders.
- Evaluaties toevoegen.
  - Door docenten.
  - Door bedrijven.

## Tables

- Gebruikers
  - Beheerders
  - Studenten
  - Docenten
  - Bedrijf medewerkers
    - Bedrijven
- Stagevoorstellingen
- Stage Evaluaties

### Users

Velden:
